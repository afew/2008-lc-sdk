Compile 순서

1. LnUtil
2. LnD3D
3. LnMdl

4. MapHF


1. 라이브러리 컴파일
DirectX, MsXML SDK를 설치해야 되는데 개발관련SDK 폴더에 헤더파일과 라이브러리 파일
이 있습니다.

2. 각각 컴파일해도 되고, MapHF프로젝트를 열어 한꺼 번에 컴파일 해도 됩니다. Alt+B,
 Alt+U, Alt+R 순으로 하면(VC 6.0의 배치빌드입니다.) 컴파일 에러가 중간에 발생하는데
 컴파일이 끝날 때 까지 기다리세요. 이것은 Model 프로젝트가 Util의 헤더파일을 참조하기
 때문입니다. 다시 한 번 Alt+B, Alt+U, Alt+R 순으로 하면 Util의 프로젝트가 컴파일되어서
 모델도 컴파일이 되어 전체적으로 에러가 없이 완전하게 컴파일이 됩니다.

3. 컴파일을 완료하면 LnInclude폴더에 10개의 헤더파일들이 복사가 됩니다. 디버그
모드로 컴파일한 경우에는 _.lib, 릴리즈 모드의 경우 .lib로 구분했습니다.

4. 실행 파일 컴파일
디버그 모드는  _Exec/blame_d.exe, 릴리즈 모드는 _Exec/blame.exe로 컴파일 됩니다.


5. 예제 열기
실행을 하고 메뉴의 Open... -> Common Dialog -> Map 폴더 -> *.bod파일을 선택합니다.

6. 메인 윈도우 카메라
	이동: W-key: 전진, S-key: 후진, A-key: 왼쪽이동, D-key: 오른쪽 이동. Wheel-Zoom
	회전: 마우스 오른쪽 버튼 누르고 마우스를 움직이면 회전.

7. Work윈도우 카메라
	+(add), -(sub). Work 다이얼로그가 활성화 되어야 작동합니다.
