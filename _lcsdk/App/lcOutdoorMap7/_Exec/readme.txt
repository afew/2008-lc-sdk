실행파일
 ./blame.exe		// Debug Mode Build
 ./blame_d.exe		// Release Mode Build

지형예제
 ./map/grave01.bod~grave03.bod

지형 모델
 ./Model

지형 관련 자료구조
 ./data

텍스춰
 ./Texture


메인 윈도우 카메라:
	이동: W-key: 전진, S-key: 후진, A-key: 왼쪽이동, D-key: 오른쪽 이동. Wheel-Zoom
	회전: 마우스 오른쪽 버튼 누르고 마우스를 움직이면 회전.

Work윈도우 카메라:
	Work 다이얼로그가 활성화 되어야 작동. +(add), -(sub) Key로 Zoom in/out동작


지형 오브젝트 삭제:
	메인 윈도우 메뉴바에서 Show Bound Box가 체크 되고 작업윈도우의 오브젝트 버튼을 누룬
	후 마우스로 클릭해서 오브젝트 선택-->Delete 키를 누르면 해당 오브젝트 삭제


브러쉬 기능:
	지형높낮이, 텍스춰 혼합, 오브젝트 배치에서 Left Ctrl키를 누르고 있으면 작동 됨
